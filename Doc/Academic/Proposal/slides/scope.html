SentriZK Framework

Project Scope & Capabilities

Target User
SME Employees
Tailored for employees in Small and Medium Enterprises (SMEs) who require secure internal collaboration tools without the complexity of enterprise suites.

ZKP Authentication

Password-less login using Groth16 zk-SNARKs. Proves identity without sending secrets to the server.

On-Device AI

Uses TensorFlow Lite (Isolation Forest) to detect behavioral anomalies locally, preserving privacy.

Privacy & Encryption

AES-256 encryption with a non-custodial server architecture. The server acts only as a verifier.
