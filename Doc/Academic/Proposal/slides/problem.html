Introduction

Problem Statement

01

Many organizations rely on internal chat platforms like Slack or Microsoft Teams, which use centralized servers to store hashed credentials. Real-world incidents, such as the 2023 Slack data breach, demonstrate that attackers can exploit stolen session tokens to expose internal communications, proving that stored credentials remain a critical vulnerability even when multi-factor authentication is in place.

02

While end-to-end encryption secures message content, it simultaneously creates a blind spot by preventing servers from analyzing chats for threats. For instance, in 2025, attackers hijacked expired Discord invite links to deliver malware undetected within normal chat flows, highlighting the urgent need for on-device AI anomaly detection to identify risky behavior locally without breaking encryption.
