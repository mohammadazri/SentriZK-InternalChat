Introduction

Problem Statement

01

Many organizations rely on internal chat platforms like Slack or Microsoft Teams, which use centralized servers to store hashed credentials. Real-world incidents, such as the 2023 Slack data breach, demonstrate that attackers can exploit stolen session tokens to expose internal communications.

02

While end-to-end encryption secures content, it creates a blind spot. For instance, in 2025, attackers hijacked expired Discord invite links to deliver malware undetected within normal chat flows, highlighting the urgent need for On-Device AI to identify risky behavior without breaking encryption keys.
